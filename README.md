	1. Adult Data Set
	http://archive.ics.uci.edu/ml/datasets/Adult

	http://archive.ics.uci.edu/ml/datasets/Motion+Capture+Hand+Postures#
	
	download the Postures.zip file
	
	MUST BE NAMED: Postures.csv
